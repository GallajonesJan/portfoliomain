AI-Elderly Health Monitoring System

Real-time Health Tracking with AI-Powered Risk Assessment

Project Overview

The AI-Elderly Health Monitoring System is an intelligent, non-invasive health tracking platform designed specifically for elderly patients aged 40 and above. This system combines IoT hardware with machine learning to provide real-time vital sign monitoring and intelligent health risk assessment.

The system integrates ESP32 microcontrollers with MAX30102 sensors to continuously monitor heart rate and blood oxygen saturation (SpO₂). Data is transmitted to a cloud-based backend where machine learning models analyze vital signs and predict alert levels, enabling early detection of potential health issues.

Key Features

  • Real-time heart rate and SpO₂ monitoring using MAX30102 sensor
  • Age-adjusted health risk assessment with ML-powered alert system
  • Automatic background data synchronization via ESP32
  • Interactive dashboard with real-time vital sign visualization
  • Historical health data tracking with Chart.js graphs
  • AI-powered chatbot (AlertoBot) for health guidance using GPT-4
  • JWT-based secure authentication and user management
  • Three-tier alert system (Normal, Warning, Critical)
  • Responsive design with dark mode support
  • MySQL database with bcrypt password encryption

Technologies Used

ESP32 MAX30102 Sensor Python Flask Machine Learning (XGBoost) MySQL HTML/CSS/JavaScript Chart.js OpenAI GPT-4 JWT Authentication Flask-CORS bcrypt ngrok

System Architecture

The system follows a three-tier architecture with clear separation of concerns:

Hardware Layer (IoT)

  • ESP32 Microcontroller: Manages sensor communication and WiFi connectivity
  • MAX30102 Sensor: Captures heart rate, SpO₂, IR, and RED light data
  • Real-time Data Transmission: Sends readings every 3 seconds to backend

Backend Layer (Python Flask)

  • RESTful API: Handles authentication, data storage, and retrieval
  • ML Alert System: XGBoost classifier with age-adjusted risk assessment
  • Database Management: MySQL with normalized schema for users and readings
  • JWT Security: Token-based authentication with bcrypt password hashing
  • ESP32 Proxy: Bridges HTTP requests between frontend and ESP32 device

Frontend Layer (Web Application)

  • Dashboard: Real-time vital sign display with Chart.js visualizations
  • Health Logs: Historical data table with automatic refresh
  • Device Management: ESP32 connection controls with background sync
  • AI Assistant: GPT-4 powered chatbot for health inquiries
  • User Settings: Profile management with secure password updates

Machine Learning Model

The system employs an XGBoost classifier trained on 1,800 synthetic samples with realistic medical parameters:

Model Specifications

  • Algorithm: XGBoost (400 estimators, max depth 6)
  • Input Features: Heart Rate, SpO₂, Age
  • Output Classes: Normal (0), Warning (1), Critical (2)
  • Age-Adjusted Thresholds: Dynamic HR/SpO₂ limits based on patient age
  • Training Accuracy: ~95% with cross-validation
  • Fallback System: Rule-based alerts if ML model fails

Technical Challenges & Solutions

Challenge 1: Real-time Data Synchronization
Solution: Implemented ESP32 background service with automatic reconnection and 3-second polling intervals to ensure continuous data flow even when the device page is closed.

Challenge 2: Cross-Origin Communication
Solution: Deployed Flask backend with ngrok tunneling and Flask-CORS to enable secure communication between local ESP32, cloud backend, and web frontend.

Challenge 3: Age-Specific Health Assessment
Solution: Developed age-adjusted ML model with dynamic thresholds that account for physiological changes in elderly patients (40-90 years).

Impact & Results

The AI-Elderly system successfully demonstrates the potential of combining IoT, machine learning, and modern web technologies to create practical healthcare solutions. Key achievements include:

  • 95% ML model accuracy in classifying health alert levels
  • Sub-3-second latency from sensor reading to dashboard display
  • Seamless background synchronization with automatic reconnection
  • Responsive interface supporting both mobile and desktop devices
  • Secure authentication system protecting sensitive health data

This project serves as a proof-of-concept for affordable, accessible health monitoring systems specifically designed for elderly care, with potential applications in home care, assisted living facilities, and remote patient monitoring programs.
